# Public Assets Folder

This folder contains static assets that are served directly by the application.

## CV/Resume

Place your CV/resume PDF file in this folder with the name `cv.pdf` to enable the download functionality on the website.

### Instructions:

1. Place your CV PDF file in this folder
2. Rename it to `cv.pdf`
3. The download button in the Hero section will automatically link to this file

**File location:** `/public/cv.pdf`

**Download URL:** `https://yourdomain.com/cv.pdf`

## Other Assets

You can also place other static files here such as:
- Images
- Documents
- Favicon
- robots.txt
- sitemap.xml

All files in this folder will be served from the root URL of your website.
